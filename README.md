# ✋ Hand2Voice – Gesture-to-Speech Translator

## 📌 Overview

Hand2Voice is a machine learning-based project that converts **sign language gestures into human-readable text and speech**.
The goal is to bridge the communication gap between **hearing-impaired individuals and others** using real-time gesture recognition.

---

## 🚀 Features

* 🎥 Gesture recognition using video dataset
* 🧠 Trained deep learning model for sign classification
* 🔊 Converts recognized gestures into speech
* 📱 Designed to be integrated with an Android application
* ⚡ Real-time processing capability (future scope)

---

## 🧠 Model Details

* Model trained using gesture video dataset
* Supports multiple sign classes (custom dataset)
* Outputs predicted gesture labels
* Can be extended for real-time webcam input

## ⚙️ How It Works

1. Collect gesture video dataset
2. Preprocess frames and extract features
3. Train deep learning model
4. Predict gesture from input
5. Convert prediction into text/speech
